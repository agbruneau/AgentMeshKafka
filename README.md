# EDA-Lab

**EDA-Lab** est une application académique de simulation d'interopérabilité dans un écosystème d'entreprise fondé sur l'architecture événementielle (EDA - Event Driven Architecture).

Ce projet permet d'apprendre et d'expérimenter l'implémentation et l'exploitation des patrons d'architecture EDA à travers une simulation automatisée de domaines métier interconnectés (Bancaire, Assurance Personne, Assurance Dommage).

## Documentation

Ce repository contient l'ensemble de la documentation et des spécifications du projet :

- **[PDR.md](./PDR.md)** (Product Definition Record) : Spécifications complètes du produit, domaines métier, événements et architecture.
- **[PLAN.md](./PLAN.md)** : Plan d'implémentation détaillé étape par étape.
- **[TODO.md](./TODO.md)** : Liste de tâches exhaustive et suivi de la progression.
- **[CLAUDE.md](./CLAUDE.md)** : Guide et contexte pour l'assistant IA Claude.
- **[CHANGELOG.md](./CHANGELOG.md)** : Journal des modifications du projet.
